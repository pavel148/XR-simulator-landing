# XR‑тренажёр — сайт инженерного решения

Это статический одностраничный сайт для заявки на премию «Вызов».

## Как открыть локально
Откройте файл `index.html` в браузере.

## Как получить публичную ссылку
Самый быстрый вариант:
1. Зайдите на https://app.netlify.com/drop
2. Перетащите всю папку `xr_engineering_site` в окно загрузки.
3. Netlify выдаст публичную ссылку вида `https://...netlify.app`.
4. Эту ссылку можно вставить в поле заявки: «ссылка на сайт инженерного решения / материал апробации».

Альтернатива: GitHub Pages. Создайте репозиторий, загрузите эти файлы в корень и включите Pages в настройках репозитория.

## Что желательно проверить перед публикацией
- заменить или добавить номер выданного патента, если он уже получен;
- добавить телефон, если он действительно должен быть публичным;
- добавить PDF‑отзыв/письмо заинтересованности отдельными файлами, если комиссия принимает документы по ссылке;
- заменить формулировки «заявка на полезную модель» на «патент», только если патент официально выдан.
